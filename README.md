## 项目运行

```
git clone https://github.com/artdong/ionic3-awesome

cd ionic3-awesome 

npm install

ionic serve (运行开发模式)

浏览器打开: http://localhost:8080

ionic cordova run build android (安卓打包发布)

ionic cordova run build ios (安卓打包发布)
```
